My Project On Console

In this project I am using Visual Studio Code which is an offline IDE software.

Console screen is used for when people are programming a project. They use it so that when they need to debug their project they can easily go to the console screen and fix the problems.

To see the console screen here are the instructions:

1.Right Click

2.Press 'Inspect'

3.On your right a screen will pop up, at the top you will see an option 'Console'

4.Click on 'Console' and view the console screen!
